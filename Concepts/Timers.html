setTimeout: Runs code after a specified period of time.

var timerID = window.setTimeout(someFunction, delayInMilliseconds);
window.clearTimeout(timerID);

setInterval: Runs code repeatedly with a specified delay between each repeat.

var timerID = window.setInterval(someFunction, delayInMilliseconds);
window.clearTimeout(timerID);

requestAnimationFrame: Request a repaint operation and runs code as part of the repaint. i.e. Running the loop continuously:

var timerID = requestAnimationFrame(someFunction);
cancelAnimationFrame(timerID);
